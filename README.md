## Run Locally

Clone the project

```bash
  git clone https://github.com/LuizSSampaio/finance-app.git
```

Go to the project directory

```bash
  cd finance-app
```

Install dependencies

```bash
  flutter pub get
```

Run the App

```bash
  flutter run
```
